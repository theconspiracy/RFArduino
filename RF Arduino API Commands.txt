RF Arduino API Commands



--LED Methods--

Blink an LED
module, method, pin, blink count, milliseconds

led,blink,13,4,500


Blink LED by Custom Pattern
module, method, pin, count, pattern with - seperator

led,customBlink,7,10,100-200




--Button Methods--

Setup button for use
module, method, empty param (use 0), empty param (use 0), pins

button,setup,0,0,2-3
